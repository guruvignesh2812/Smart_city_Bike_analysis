## 📌 Overview
This project is a Power BI dashboard designed to analyze bike-sharing systems across multiple cities. It provides insights into bike stand capacity, station distribution, availability, and usage patterns to help identify demand gaps and optimize operations.

## 🎯 Key Objectives
- Analyze bike availability vs capacity
- Identify high-demand and underutilized stations
- Compare city-wise bike infrastructure
- Understand station distribution trends
- Visualize geographical spread of bike stations

## 📊 Dashboard Features
    
### 1. KPI Cards
- Total Stations: 3K  
- Bike Stand Capacity: 57K  
- Avg Available Bikes per Station: 6.88
- 

### 2. City-Level Analysis
- Compare total capacity vs available bikes
- Identify cities with:
  - High unused capacity
  - Low bike availability (high demand)

### 3. Station-Level Insights
- Analyze top stations by capacity
- Track station throughput
- Detect bottlenecks & underperforming stations

### 4. Distribution Analysis
- Number of stations across cities
- Capacity contribution by top cities
- Payment type distribution:
  - Online vs Cash usage

### 5. Geo Visualization
- Map view showing bike station locations
- Identify regional density and coverage gaps

## 📈 Key Insights
- Cities like Lyon & Bruxelles-Capitale have the highest capacity and availability
- Some cities show high capacity but low availability → demand mismatch
- Stations with low availability indicate high usage or shortage
- Majority of users prefer online payment (~88%)
